Un "{}\+Entorno"{} en Python es el contexto en el que se ejecuta un programa de Python que consta de un intérprete y una cantidad de paquetes instalados.\hypertarget{md__c_1_2_repositorio_t_i_f_2_software_2_trabajo_integrador_final___fisio_2deps_2_r_e_a_d_m_e_autotoc_md3}{}\doxysubsection{\texorpdfstring{Entorno global\+:}{Entorno global\+:}}\label{md__c_1_2_repositorio_t_i_f_2_software_2_trabajo_integrador_final___fisio_2deps_2_r_e_a_d_m_e_autotoc_md3}
Por defecto, cualquier intérprete de Python instalado se ejecuta en su propio entorno global. Por ejemplo, si se ejecuta el comando python o python3 en alguna terminal (dependiendo de cómo se haya instalado Python), se estaría ejecutando en el entorno global de ese intérprete. Cualquier paquete que se instale o desinstale afecta el entorno global y por tanto todos los programas que se ejecutan en él.

En Python, es una buena práctica crear un entorno específico de un espacio de trabajo o entorno virtual local.\hypertarget{md__c_1_2_repositorio_t_i_f_2_software_2_trabajo_integrador_final___fisio_2deps_2_r_e_a_d_m_e_autotoc_md4}{}\doxysubsection{\texorpdfstring{Entorno virtual local\+:}{Entorno virtual local\+:}}\label{md__c_1_2_repositorio_t_i_f_2_software_2_trabajo_integrador_final___fisio_2deps_2_r_e_a_d_m_e_autotoc_md4}
Este tipo de entorno te permite instalar módulos de python sin afectar otros entornos aislando la instalación de los módulos del espacio de trabajo.\hypertarget{md__c_1_2_repositorio_t_i_f_2_software_2_trabajo_integrador_final___fisio_2deps_2_r_e_a_d_m_e_autotoc_md5}{}\doxysubsection{\texorpdfstring{Creación de entorno virtual local en VSCode}{Creación de entorno virtual local en VSCode}}\label{md__c_1_2_repositorio_t_i_f_2_software_2_trabajo_integrador_final___fisio_2deps_2_r_e_a_d_m_e_autotoc_md5}

"{}requirements.\+txt"{} es el archivo que guardará los nombres de los módulos instalados para nuestro proyecto, así como las versiones de los mismos.

Para generar este archivo en la carpeta {\ttfamily deps/} ejecutar en la terminal de VSCode\+:

{\ttfamily pip freeze \texorpdfstring{$>$}{>} deps/requirements.\+txt}.
